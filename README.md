    2. Open config Apache XAMPP

        DocumentRoot "<yourpath_xampp_installed>/htdocs/public"
        <Directory "<yourpath_xampp_installed>/htdocs/public">
              <Directory "<yourpath_xampp_installed>/htdocs/public">
